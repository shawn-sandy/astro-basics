AstroKit is a simple and easy to use starter kit for  building accessible and performant websites with Astro.  It includes features like MDX, syntax highlighting, and more. This site is built with Astro, so you can use it as a reference for building your own site. Find out more about Astro by visiting the [Astro docs](https://docs.astro.build/)

 For additional info on installation and setup, creating, customizing, etc please read the docs (below)!
