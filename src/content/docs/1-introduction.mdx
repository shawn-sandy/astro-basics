## Welcome to AstroKit

 This is an introductory post for AstroKit, a starter kit for building blogs and documentation sites with Astro.

 AstroKit is a starter kit for building blogs and documentation sites with Astro. It includes features like MDX, syntax highlighting, and more. This site is built with Astro, so you can use it as a reference for building your own site. Find out more about Astro by visiting the [Astro docs](https://docs.astro.build/)


AstroKit Features 

- Easy blogging with MDX
- Automatic route generation for posts
- Frontmatter support
- Syntax highlighting
- Automatic image optimization**
- Pagination for long-form content
- Breadcrumds for navigation
- Tags and categories for filtering posts
- Easy customization with components
- Accessibility built in 
- SEO optimized

## Getting started

To get started with AstroKit, you can clone the repository from GitHub and run the following commands:

```bash 
# Clone the repository
git clone https://github.com/withastro/astrokit.git my-blog
# Navigate to the directory
cd my-blog
# Install dependencies
npm install
# Start dev server
npm run start
```

This will start a development server at `http://localhost:3000` where you can see the site in action. You can start writing your own posts by adding MDX files to the `src/posts` directory. The posts will automatically be generated as routes.
